Linguagem C#

C# é uma linguagem de programação, multiparadigma, de tipagem forte. É uma ferramenta muito popular para criação de jogos e portanto, é uma ótima opção para qualquer programador que pretenda entrar no setor de desenvolvimento de jogos ou para qualquer pessoa interessada em realidade virtual.

Motor de Jogos Unity

O Unity é uma ferramenta que permite criar jogos para diversas plataformas (PC, console, mobile, VR e AR) utilizando um editor visual e programação através de scripting, oferecendo aos utilizadores ferramentas profissionais, capazes de preencher os requisitos de qualquer jogo.

Fruity loops

FL Studio é uma estação de trabalho de áudio digital desenvolvido pela empresa belga Image-Line. Atualmente é um dos softwares mais usados em produções musicais no mundo. Ficando em 4° lugar com 13.63%.

Aseprite

Aseprite é um editor de imagens disponível na fonte, projetado principalmente para desenho e animação de pixel art. Ele roda em Windows, macOS e Linux, e possui diferentes ferramentas para edição de imagens e animações como camadas, frames, suporte a tilemap, interface de linha de comando, scripts Lua, entre outros.
